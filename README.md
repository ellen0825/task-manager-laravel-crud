# Менеджер задач

Простое CRUD-приложение для управления задачами, написанное на Laravel 12.

## Стек

- **PHP** 8.2+
- **Laravel** 12
- **SQLite** (база данных по умолчанию)
- **Blade** + **Bootstrap 5**

## Функциональность

- Список задач с поиском по названию и фильтрацией по статусу
- Создание, редактирование, просмотр и удаление задач
- Три статуса задачи: **Новая**, **В процессе**, **Завершённая**
- Валидация форм
- Пагинация

## Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd <repo-folder>

# 2. Установить зависимости
composer install

# 3. Создать файл окружения и сгенерировать ключ
cp .env.example .env
php artisan key:generate

# 4. Создать базу данных и выполнить миграции
touch database/database.sqlite
php artisan migrate

# 5. Запустить сервер
php artisan serve
```

Приложение будет доступно по адресу: [http://localhost:8000](http://localhost:8000)

## Структура задачи

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | integer | Первичный ключ |
| `title` | string | Название задачи |
| `description` | text | Описание (необязательно) |
| `status` | enum | `new` / `in_progress` / `done` |
| `created_at` | timestamp | Дата создания |
| `updated_at` | timestamp | Дата обновления |
